Computer Hardware

MOTHERBOARD

A motherboard is a printed circuit board containing the principal components of a computer or other device, with connectors into which other circuit boards can be slotted. A motherboard allows the CPU, RAM, and all other computer hardware components to communicate with each other.

CPU

A CPU is central processing unit. The CPU is the brains of the computer where most calculations take place. A CPU performs arithmetic and logical operations, and the control unit (CU), which extracts instructions from memory and decodes and executes them. It's like the brains of the computer.(Google)

RAM

RAM random access memory. RAM is a type of computer memory that can be accessed randomly. RAM is the memory of your computer so it saves whatever your doing.

Hard Drive

A hard drive is a high-capacity, self-contained storage device containing a read-write mechanism plus one or more hard disks, inside a sealed unit. Also called hard disk drive. A hard drive is used for storing and retrieving digital information using rapidly rotating disks.(Google)

Power Supply

A power supply is the thing that supplies elctric energy toan electric load. For computers is a big box with a fan and for a laptop is a cable or the charger. The primary function of a power supply is to convert one form of electrical energy to another.

Different Components Communicating To Each Other

Computer components are connected to each other via thin connections known as 'bus' on the motherboard. Since all the componets are connected together, that is the way they communicate.
